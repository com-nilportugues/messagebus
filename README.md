# MessageBus 

Concrete implementations of the MessageBus:

 - CommandBus
 - QueryBus

EventBus is not included.  
 
 
## Installation

Add in your `pom.xml` file the jitpack.io repositories:

```xml
<repositories>
  <repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
  </repository>
</repositories>
```
  
Now add the package as a dependency: 

```xml
<dependencies>		
  <dependency>
    <groupId>com.github.com-nilportugues</groupId>
    <artifactId>messagebus</artifactId>
    <version>${messagebus.version}</version>
  </dependency>
</dependencies>  
```
